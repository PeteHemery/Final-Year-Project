Final-Year-Project by PeteHemery

Undergraduate Dissertation for University of the West of England,
BSc Computer Systems Integration 2011/2012.

Educational introduction into using the Multicore Parallax Propeller Microcontroller for Audio Analysis and Graphical Representation.

Download

You can download all files related to this project in either zip or tar formats.

Download the Propeller Tool archived versions of
the Graphical Tuner Demonstration
or
the Spectrograph Demonstration

Download the Dissertation

Dependencies

Parallax Propeller Demo Board.
Windows/Linux/Mac.

Install

For Linux, Mac OSX or Windows:
bst - Brad's Spin Tool

For Windows:
The Propeller Tool - Parallax Inc.

Download and install the IDE for your platform.
Connect the Propeller.
The README file will show which is the Top Object File.
Open that object, and run.
